# image-windows-7

This repository defines a Docker image based on Windows 7. It provides a Windows 7 operating system environment within a Docker container, primarily intended for compatibility with legacy applications or specific testing scenarios that require the Windows 7 operating system.

## Key Features and Structure:

*   **Dockerfile**: Contains instructions for building the Windows 7 Docker image.
*   **Base OS**: Utilizes Windows 7, supporting older software and systems.
*   **Legacy Application Support**: Enables the execution of applications that may not be compatible with newer Windows versions.

## Role in CyberRangeCZ:

The `image-windows-7` repository is valuable for maintaining compatibility with older systems and applications, ensuring that legacy components or specific testing requirements can be met within the CyberRangeCZ environment.
