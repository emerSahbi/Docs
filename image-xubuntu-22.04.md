# image-xubuntu-22.04

This repository defines a Docker image based on Xubuntu 22.04 LTS (Jammy Jellyfish). Xubuntu is a lighter-weight derivative of Ubuntu, utilizing the Xfce desktop environment. This image offers a stable and resource-efficient graphical Linux environment suitable for development, testing, or applications that benefit from a GUI but have limited resource requirements.

## Key Features and Structure:

*   **Dockerfile**: Contains instructions for building the Xubuntu 22.04 LTS Docker image.
*   **Base OS**: Utilizes Xubuntu 22.04 LTS, providing a modern and stable Linux environment with the Xfce desktop.
*   **Lightweight GUI**: Offers a graphical user interface that is suitable for interactive development and testing workflows.

## Role in CyberRangeCZ:

The `image-xubuntu-22.04` repository provides a current, stable, and graphical Linux environment that is lighter on resources than a full Ubuntu desktop. This is beneficial for development or testing workflows requiring a GUI within the CyberRangeCZ platform.
