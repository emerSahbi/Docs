# image-opnsense

This repository defines a Docker image based on OPNsense, an open-source firewall and routing platform. This image is likely used to create network appliances, test network configurations, or deploy firewall services within the CyberRangeCZ environment.

## Key Features and Structure:

*   **Dockerfile**: Contains instructions for building the OPNsense Docker image.
*   **Base OS**: Utilizes OPNsense, known for its robust firewall, VPN, and network management capabilities.
*   **Network Appliance**: Designed to function as a virtual firewall or router.

## Role in CyberRangeCZ:

The `image-opnsense` repository provides a dedicated firewall and network management solution. It is crucial for securing network segments, managing traffic, and establishing secure VPN connections within the CyberRangeCZ platform.
