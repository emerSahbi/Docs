# images-wiki

This repository is dedicated to hosting the wiki documentation for the Docker images used within the CyberRangeCZ project. It serves as a central knowledge base for understanding the purpose, usage, and configurations of different base images.

## Key Features and Structure:

*   **Wiki Content**: Contains markdown files or other wiki-formatted content detailing each Docker image.
*   **Image Information**: Provides explanations for base operating systems, their intended use cases, and any specific customizations.
*   **Centralized Documentation**: Acts as a single source of truth for image-related documentation, ensuring consistency and accessibility.

## Role in CyberRangeCZ:

The `images-wiki` repository plays a vital role in maintaining clear and accessible documentation for all Docker images. This helps developers and operators quickly understand and correctly utilize the foundational components of the CyberRangeCZ platform.
