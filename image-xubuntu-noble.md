# image-xubuntu-noble

This repository defines a Docker image based on Xubuntu Noble Numbat (24.10). Xubuntu is a lighter-weight derivative of Ubuntu, known for its use of the Xfce desktop environment. This image provides a stable and resource-efficient graphical Linux environment, suitable for development, testing, or applications that benefit from a GUI but have limited resource requirements.

## Key Features and Structure:

*   **Dockerfile**: Contains instructions for building the Xubuntu Noble Numbat Docker image.
*   **Base OS**: Utilizes Xubuntu Noble Numbat (24.10), offering a recent Ubuntu-based distribution with the Xfce desktop.
*   **Lightweight GUI**: Provides a graphical user interface that is suitable for interactive development and testing workflows.

## Role in CyberRangeCZ:

The `image-xubuntu-noble` repository provides a current, stable, and graphical Linux environment that is lighter on resources than a full Ubuntu desktop. This is beneficial for development or testing workflows requiring a GUI within the CyberRangeCZ platform.
