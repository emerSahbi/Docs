# image-rocky-8

This repository defines a Docker image based on Rocky Linux 8. Rocky Linux is a community enterprise operating system, designed to be a bug-for-bug compatible downstream rebuild of RHEL (Red Hat Enterprise Linux). This image serves as a stable, enterprise-grade operating system environment for applications and services.

## Key Features and Structure:

*   **Dockerfile**: Contains instructions for building the Rocky Linux 8 Docker image.
*   **Base OS**: Utilizes Rocky Linux 8, providing an enterprise-grade, stable Linux environment.
*   **RHEL Compatibility**: Offers binary compatibility with RHEL, making it suitable for enterprise applications.

## Role in CyberRangeCZ:

The `image-rocky-8` repository provides a robust and stable enterprise Linux environment, suitable for hosting critical services or for development and testing scenarios that require RHEL-like compatibility.
