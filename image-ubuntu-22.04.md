# image-ubuntu-22.04

This repository defines a Docker image based on Ubuntu 22.04 LTS (Jammy Jellyfish). This Long Term Support version provides a stable and widely-supported Linux environment, making it a common choice for development, testing, and deployment of various applications and services.

## Key Features and Structure:

*   **Dockerfile**: Contains instructions for building the Ubuntu 22.04 LTS Docker image.
*   **Base OS**: Utilizes Ubuntu 22.04 LTS, known for its extensive package repository and long-term support.
*   **General Purpose**: Suitable for a wide range of applications due to its widespread adoption and compatibility.

## Role in CyberRangeCZ:

The `image-ubuntu-22.04` repository provides a reliable and up-to-date Ubuntu environment, serving as a fundamental base image for many components within the CyberRangeCZ platform that require a modern Linux distribution.
