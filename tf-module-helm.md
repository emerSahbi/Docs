# tf-module-helm

This repository contains Terraform modules specifically designed for managing Helm chart deployments within Kubernetes environments. It enables declarative management of Helm releases, simplifying the deployment and lifecycle management of applications packaged as Helm charts.

## Key Features and Structure:

*   **Helm Chart Management**: Provides Terraform resources to define, install, upgrade, and manage Helm releases.
*   **Kubernetes Integration**: Seamlessly integrates with Kubernetes clusters managed by Terraform.
*   **Declarative Deployment**: Allows for the declarative specification of Helm chart configurations and dependencies.

## Role in CyberRangeCZ:

The `tf-module-helm` repository enhances the deployment automation capabilities for the CyberRangeCZ platform. It allows infrastructure teams to manage application deployments on Kubernetes using Helm charts in a consistent, version-controlled, and automated manner.
