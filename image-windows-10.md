# image-windows-10

This repository defines a Docker image based on Windows 10. It provides a Windows 10 operating system environment within a Docker container, suitable for running Windows-specific applications, testing software compatibility, or conducting development tasks that require the Windows 10 ecosystem.

## Key Features and Structure:

*   **Dockerfile**: Contains instructions for building the Windows 10 Docker image.
*   **Base OS**: Utilizes Windows 10, offering a familiar desktop Windows environment.
*   **Windows Applications**: Enables the execution of applications that are exclusively designed for or perform best on Windows 10.

## Role in CyberRangeCZ:

The `image-windows-10` repository offers a critical environment for scenarios requiring Windows 10 compatibility, such as testing cross-platform applications, running Windows-specific tools, or supporting hybrid infrastructure within the CyberRangeCZ platform.
