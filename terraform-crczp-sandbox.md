# terraform-crczp-sandbox

This repository contains Terraform code used to define and manage the infrastructure for the CyberRangeCZ sandbox environment. It automates the provisioning and configuration of cloud resources necessary for creating isolated testing and training environments.

## Key Features and Structure:

*   **Terraform Modules**: Utilizes Terraform modules to define reusable infrastructure components for sandbox deployments.
*   **Infrastructure as Code (IaC)**: Manages the deployment and configuration of cloud resources (e.g., VMs, networks, storage) through declarative code.
*   **Sandbox Environment Provisioning**: Automates the creation of isolated, on-demand sandbox environments for users.
*   **Resource Management**: Enables efficient management and teardown of sandbox resources.

## Role in CyberRangeCZ:

The `terraform-crczp-sandbox` repository is crucial for providing users with flexible and isolated sandbox environments. It ensures that these environments are provisioned and managed consistently and automatically, supporting training and development activities.
